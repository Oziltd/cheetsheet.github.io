CSS

When We explore CSS it allows us to set the basic foundations to the design of the webpage. Allowing us to edit images, fonts size and weight, space lining literally most things you see on the screen.

Typography

When we talk about fonts we explore the font they use and the weight of the font so how bold or italic or thin
font-family refers to a family of fonts available for selected elements
font-size sets the text size because texts size can be of different values
letter-spacing allows control on horizontal spacing between characters in selected element
line-height which sets the line spacing within paragraphs
text-align how you want to position a line of text (Left, Centre, Right)
